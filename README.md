## REQUIREMENTS  
* git  
* [yay](https://github.com/Jguer/yay)  

## HOW TO  
Install git  
```
sudo pacman -Syyu  
sudo pacman -S git  
```  
Install yay : [Click Here](https://github.com/Jguer/yay#Installation)  
Install Config  
```
git clone https://github.com/wulan17/zsh-config.git  
cd zsh-config  
bash setup.sh  
```
